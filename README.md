## About

A fully responsive, 3D interactive personal portfolio website built to showcase my skills, projects, internships, and certifications in Data Science and Engineering. Designed with a futuristic neon aesthetic featuring animated particle networks, 3D card effects, and smooth scroll animations.

- 3D animated hero section with particle network background
- Smooth scroll-reveal animations on all sections
- 3D tilt effect on project cards

- Fully responsive design for desktop and mobile
